# Python — Sistema di Gestione degli Elementi (SGE) ⚗️

> Sistema completo per la gestione di una collezione di elementi, con funzionalità di registrazione, ricerca, filtraggio avanzato e statistiche — sviluppato in Python puro.

**IT** | Progetto Python realizzato nell'ambito del Master in Data Science, Analytics e AI @ Start2Impact University.

---

## Obiettivo / Objective

**IT:** Sviluppare un sistema completo per la gestione di una collezione di elementi, implementando registrazione, visualizzazione, ricerca, filtraggio avanzato e statistiche tramite Python puro — senza dipendenze da database esterni.

**EN:** Build a complete management system for a structured data collection, implementing registration, display, search, advanced filtering and statistics using Python's native data structures and no external dependencies.

---

## Funzionalità / Features

Funzionalità base:
- Registrazione di un nuovo elemento con attributi specifici
- Visualizzazione di tutti gli elementi registrati
- Ricerca per uno o più attributi

Funzionalità aggiuntive:
- **Statistiche sulla collezione:** conteggio totale, distribuzione per categoria, elemento più frequente/recente
- **Filtraggio avanzato:** ricerca con condizioni multiple simultanee (es. tutti gli elementi che soddisfano criterio A e criterio B contemporaneamente)
